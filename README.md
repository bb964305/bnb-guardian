## 🎯 这是什么？

BNB Guardian 是一个 OpenClaw Agent Skill，把你的 AI Bot 变成一个**全自动的币安交易管家**。

- 🤖 **装上就跑** — 首次引导配置后，全程自动运行
- 📡 **全盘监控** — 币安全部交易对实时扫描，不漏任何机会
- 🧠 **AI 决策** — 多维数据综合分析，自主判断买卖
- 🛡️ **智能风控** — 止损、熔断、黑天鹅保护，保住本金
- 📈 **自我进化** — 每日复盘，策略淘汰，参数自优化

**用户只需要做一件事：安装 Skill，提供 API Key。剩下的全交给 AI。**

## ✨ 功能全景

### 📡 市场监控
- 币安全部交易对实时扫描（现货 + U本位合约 + 币本位合约）
- 价格异动检测（涨跌幅、成交量暴增、突破均线）
- 深度数据监控（多空比、持仓量、大户动向、资金费率）
- 盘口分析（买卖盘厚度、支撑压力判断）

### 📊 技术分析
- **多时间框架共振**：日线（趋势）+ 4H（信号）+ 1H（入场）
- **技术指标**：MA(7/25/99/200)、RSI(14)、MACD(12,26,9)、布林带(20,2)
- **综合评分**：-100 到 +100，结合 10 个维度加权计算
- **时段分析**：亚盘/欧盘/美盘不同策略

### 📰 多源情报
- 恐惧贪婪指数（Alternative.me）
- CoinGecko 全球数据（BTC 主导率、总市值）
- 新闻聚合分析（利好/利空判断）
- X/推特情绪分析
- 链上数据（交易所流入流出、鲸鱼转账）
- 清算数据（大规模爆仓监控）
- 币安公告（新币上线、维护、规则变更）

### 💰 交易执行
| 产品 | 能力 |
|------|------|
| **现货** | 市价/限价/OCO/冰山委托/分批建仓 |
| **合约** | 做多/做空/追踪止损/杠杆设置(1-125x) |
| **杠杆** | 借币/还币/杠杆下单 |

### 🎯 策略引擎
| 策略 | 适用场景 |
|------|---------|
| 趋势跟踪 | 明确上涨/下跌趋势 |
| 网格交易 | 横盘震荡区间 |
| 费率套利 | 资金费率异常（>0.1%）|
| 三角套利 | 三角汇率偏离 |
| 插针抄底 | RSI<15 + 暴跌 + 无实质利空 |
| DCA 定投 | 定时定额买入 |
| 移动止盈 | 单边上涨行情 |

**Agent 根据市场状态自动选择最优策略，无需人工指定。**

### 🛡️ 智能风控
- **仓位控制**：单笔≤5%、单币≤20%、保留20%现金
- **凯利公式**：半凯利计算最优仓位
- **相关性检查**：相关性>0.8的币种合计仓位≤30%
- **熔断机制**：日亏损>5%暂停24h、连亏3笔暂停4h
- **黑天鹅保护**：15分钟跌>15%立即清全部合约
- **杠杆降级**：亏损时自动降低杠杆
- **盈利保护**：盈利>5%止损收紧到保本
- **滑点保护**：偏差>1%告警、>2%暂停市价单

### 🧠 自我进化
- **每日复盘**：分析每笔交易的胜败原因
- **策略绩效追踪**：胜率、盈亏比、夏普比率
- **自动淘汰**：30天胜率<40%的策略自动暂停
- **参数自优化**：每月回测优化 RSI 阈值、MA 周期等
- **学习日志**：记录教训和规律，不犯重复错误

### 📢 币安广场自动运营
- **6 种帖子类型**：行情分析、市场情绪、交易复盘、教育科普、热点追踪、周报
- **自动互动**：专业评论、点赞、关注
- **涨粉策略**：固定时间发帖、追热点、展示实盘收益
- **热帖分析**：学习什么内容受欢迎

### 📲 推送通知
| 级别 | 场景 |
|------|------|
| 🔴 紧急 | 风控触发、爆仓风险、黑天鹅、API异常 |
| 🟡 重要 | 交易执行、止损止盈、策略切换、套利 |
| 🟢 常规 | 异动检测、新高新低、定期报告 |

定期报告：每日早报(8:00) + 晚报(20:00) + 周报(周日) + 月报

## 🚀 快速开始

### 1. 安装 OpenClaw

```bash
npm install -g openclaw
```

### 2. 安装 BNB Guardian Skill

将 `SKILL.md` 放到你的 OpenClaw workspace 的 skills 目录：

```bash
# 克隆仓库
git clone https://github.com/aa964305/bnb-guardian.git

# 复制到 OpenClaw skills 目录
cp -r bnb-guardian ~/.openclaw/workspace/skills/binance-trader
```

### 3. 启动 Bot

Bot 启动后会自动检测到新 Skill，并触发首次引导流程：

1. ⚠️ **风险告知** — 确认了解交易风险
2. 🔑 **API Key** — 提供币安 API Key（只读 + 交易，❌不要开提现）
3. ⚙️ **偏好设置** — 选择风险偏好、关注币种、交易模式
4. ✅ **完成** — Bot 开始自动运行

### 4. 坐等收通知 📱

## ⚠️ 安全须知

1. **API Key 权限**：只开读取 + 交易，**绝对不要开提现**
2. **IP 白名单**：建议设置 API Key 的 IP 白名单
3. **小仓位测试**：先用通知模式观察，确认策略有效后再开自动交易
4. **风控不可关闭**：风控规则是硬编码的，无法绕过
5. **凭证安全**：API Key 存储在 `vault/` 目录，不会上传或泄露
